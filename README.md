# EAAT2 Machine Learning Project

This repository contains the code and data used in my bachelor thesis on ligand-based machine learning for EAAT2 inhibitors and the data for EGFR 
Both the .ipynb files were used in jupyter lab to generate the data for this project.
Both the .csv files were generated and filtered using jupyter lab for the EAAT2 and EGFR target.
Most figures although presented here might not be in the final report and that's because they wouldn't have added much to the content.

This project is intended for academic and research purposes only
